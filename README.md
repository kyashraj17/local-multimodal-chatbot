# Local Multimodal Chatbot

A Streamlit-based multimodal AI chatbot that enables conversational question answering over PDF documents using vector embeddings.

## Features
- Interactive chatbot interface using Streamlit
- PDF document upload and chat
- Semantic search using ChromaDB
- Built with LangChain for LLM orchestration
- Retrieval-Augmented Generation (RAG) architecture

## Tech Stack
- Python
- Streamlit
- LangChain
- ChromaDB
- Sentence Transformers

## Usage
1. Install dependencies using `requirements.txt`
2. Run the application using:
   ```bash
   streamlit run main.py
